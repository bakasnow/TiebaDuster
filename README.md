#### 界面
![软件预览图](https://www.bakasnow.com/tuchuang/td2017.png "td2017.png")

#### 功能
1. 支持多账号多用途操作
2. 支持扫描标题、1楼内容、帖子内容
3. 支持用户黑名单、用户白名单、帖子黑名单、帖子白名单
4. 支持关键词、等级、发帖时间、是否关注等多参数关联规则
5. 关键词支持正则表达式
